## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Build for Production

```bash
npm run build
npm start
```

## Adding Projects and Internships from LinkedIn

Since LinkedIn doesn't allow automated scraping, you'll need to manually extract your data:

1. Visit your LinkedIn profile: https://www.linkedin.com/in/kanishka-sonkar-30520a243/
2. Copy your projects and internships information
3. Edit the component files directly:
   - `components/Projects.tsx` for projects
   - `components/Internships.tsx` for internships

For detailed instructions, see [LINKEDIN_DATA_GUIDE.md](./LINKEDIN_DATA_GUIDE.md)
